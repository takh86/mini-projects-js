# 🥁 Drum Kit Project

This is a beginner-friendly project built with **HTML**, **CSS**, and **JavaScript** to simulate a virtual drum kit.

Users can click on buttons or press keys to play different drum sounds.

## 🔧 Technologies Used

- HTML
- CSS
- JavaScript (DOM Events & Audio API)

## 🎯 Features

- Click or keyboard-triggered drum sounds
- Key animation when pressed
- Easy to customize with new sounds
